# WorldCup Parser

  - Generated by Express, improved by MySQL2 and CRON
  - Calculates stats of every team
  - Dinamically updating after every game in group stage (obsoleted)

### Installation

```sh
$ cd repository
$ npm i
$ npm start
```

Before the start you must update 'configs/mysql.js' file
